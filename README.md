# Matrizes Esparsas
Projeto acadêmico desenvolvido para a disciplina de Estruturas de Dados.  O sistema implementa operações com matrizes esparsas utilizando estruturas de dados dinâmicas, permitindo armazenar apenas os elementos não nulos e reduzir o consumo de memória.

# Funcionalidades
-Inserção de elementos
-Exibição da matriz
-Soma de matrizes
-Multiplicação por escalar
-Operações com estruturas dinâmicas

# Tecnologias utilizadas
-Linguagem C
-Estruturas de Dados
-Ponteiros
-Alocação Dinâmica

# Objetivos de aprendizagem
-Compreender o conceito de matrizes esparsas
-Aplicar listas encadeadas e ponteiros
-Otimizar armazenamento de dados
-Desenvolver lógica de programação
